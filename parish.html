Annotations

When I always ask myself what is the purpose of the design. A way of defining a hierarchy that will guide the user to push a button is to annotate or add strips that help define the individual user journey visually from top to bottom

It's also a good way to make sure a particular design contains everything it should. Please see below more information what each strip contains:

  • Emotion

    humans are visual creatures so often emotional connection comes in the form of an image. An image usually hooks the user and hence strongly influence their decision. A great image of the Parish Music Box playing live should peek a user's interest!

  • Description

    It should give you a validation of the emotional first impression.First confirmation in making the user's decision

  • Details

    Will give user more information to clear any doubts. The description and the details sections should be asking or answering the question is this for me? So here is this a band I want to book or check out?

  • Clarify

    This could be user reviews that will strongly influence the user to make a final decision. If it's good for others it should be good for me!

  • Action

    Once the user reaches this stage, the user should know everything he or she needs to make the final decision. Hopefully any user here will book the band or see them live!
